Readme:
===========

The program was developed in a Windows 7 environment, in the Eclipse IDE. The Programming Assignment folder contains the program files. The programs are written in java.

Compilation Instructions:
===========================


Compile the files in the following order:

1.Node.java
2.Amount.java
3.Attributes.java
4.DecisionTree.java


					javac <filename>.java

Running Instructions:
=======================

After the programs have been compiled in the order specified, please run the program by executing the file DecisionTree as it contains the main method, i.e, 

					java DecisionTree <trainingdata> <testdata>


The program should execute and the output is produced.
